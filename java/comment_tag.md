## 주석 태그 (Comment Tag) 종류

| 주석 태그      | 설명                           | 커스텀 주석 태그 여부 |
|------------|------------------------------|--------------|
| TODO       | 앞으로 해야 할 일                   | X            |
| FIXME      | 앞으로 수정되어야 할 일                | X            |
| XXX        | 추후 수정 되어야 할 일                | O            |
| BUG, DEBUG | 다른 프로그래머에게 문제가 있고 잘못된 코드를 경고 | O            |
| NOTE       | 버그가 발생한 위치                   | O            |
| TODO       | 추가 정보 또는 설명을 나타내는 주석         | O            |
